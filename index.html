Una breve descrizione del font derivato da Titillium Web

Descrizione

Il font Titillium è stato realizzato dall'Accademia di Belle Arti di Urbino e distribuito con licenza OFL 1.1[1]; è ora distribuito via Google Fonts come Titillium Web in undici diverse versioni[2].

Il font variabile Pass A38 è una derivazione del font Titillium Web (la lettera l minuscola è stata leggermente modificata per cercare di renderla facilmente distinguibile dalla I maiuscola).

Pass A38 ❦ Variabile

Come mostra la tabella, il font Pass A38 permette di riprodurre quasi tutte le varianti di Titillium Web; non è però disponibile la versione "Black" (cioè il caso con weight=900) perché non esiste il corrispondente corsivo.

Weight Tondo Corsivo
100 Thin Thin Italic
200 ExtraLight ExtraLight Italic
300 Light Light Italic
400 Regular Italic
500 Medium Medium Italic
600 SemiBold SemiBold Italic
700 Bold Bold Italic

Il font può essere utilizzato tramite un unico file in formato WOFF2 con il seguente codice CSS.

@font-face {
    font-family: "Pass A38";
    font-weight: 100 700;
    src: url("Pass_A38.woff2") format("woff2-variations");
}

Ovviamente il percorso del file deve essere adattato alla specifica situazione.

Per ottenere correttamente il corsivo con Chrome e Opera, utilizzare il seguente codice CSS.

em, i { font-variation-settings: "ital" 1; font-style:normal }

Nome

Il nome Titillium era un omaggio al compositore Arvo Pärt e alla sua tecnica tintinnabuli[3].

Poiché il font Titillium Web è stato adottato per la pubblica amministrazione italiana[4], la scelta del nome Pass A38 intende essere un omaggio all'ottava fatica di Asterix[5] per ottenere il "lasciapassare A38" („Passierschein A38“ in tedesco) nella «Casa che rende folli».

Caratteristiche

Il font è distribuito nei formati ttf e WOFF2 e sono supportate le seguenti caratteristiche.

'sups' A0123456789 → A0123456789
font-feature-settings: 'sups';
'numr' A0123456789 → A0123456789
font-feature-settings: 'numr';
'dnom' A0123456789 → A0123456789
font-feature-settings: 'dnom';
'subs' A0123456789 → A0123456789
font-feature-settings: 'subs';
'frac' A12345/67890 → A12345/67890
font-feature-settings: 'frac';
'ss01' 1234 38 → 1234 38
font-feature-settings: 'ss01';
'ss02' <- -> ^ _ → <- -> ^ _
font-feature-settings: 'ss02';
'ss03' 1234567890 → 1234567890
font-feature-settings: 'ss03';

Sempre in omaggio alla «Casa che rende folli», la caratteristica 'ss01' permette di ottenere i numeri romani dei piani e degli sportelli (solo nell'intervallo 1-9999), mentre la caratteristica 'ss02' permette di mostrare le frecce per orientarsi tra gli sportelli.

Simboli per elenchi

Il font contiene vari simboli per elenchi puntati; possono essere utilizzati in elenchi con marcatore ul tramite il seguente codice CSS (si hanno risultati diversi con Firefox rispetto a Chrome o Opera).

ul li::marker { font-family: "Pass A38"; }
